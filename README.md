# InstaClone

### Brief

#### * This is the Firebase based Inastclone with all the Feature of Instagram.
#### * in which, Users can Follow/Unfollow each other, Put/Delete Comment on eachother's Post.
#### * User also gets notified when someone Follow him/her or Commneted on his/her Post and Directly go to the Post or Users' Profile.
#### * They can Like/Dislike Post of other users and Save them as well in their Profile Section.
#### * User can Put Story visible for his/ her Followers for 24 hrs. User also delete it manually at anytime.
#### * People can Search eachother and Hastags as well.
#### * Users can also be able to See their Followers or Following list and Follow/Unfollow them from there as well.
#### * Users can Upload Existing Images or they can click a new One to Post and Stories.
